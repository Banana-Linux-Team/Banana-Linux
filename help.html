Como baixar Aplicativos do AUR

No Banana Linux já existe um instalador de pacotes do AUR e você pode utilizar-lo assim:

Para instalar você usa banana install nome-do-pacote

Para remover você usa banana remove nome-do-pacote

Para pesquisar por pacotes do AUR utilize banana search nome-do-pacote
